# Brazil E-Commerce Sales Analysis

## Project Overview
This project analyzes the **Brazilian Olist E-Commerce dataset** to understand sales trends, product performance, and delivery efficiency.

The project demonstrates **SQL analysis, Python data exploration, and Power BI dashboard development** to extract meaningful business insights.

---

## Tools & Technologies

- SQL (MySQL)
- Python
- Power BI
- Pandas
- Matplotlib
- Git
- Linux

---

Dataset used in this project:

Brazilian Olist E-Commerce Dataset
